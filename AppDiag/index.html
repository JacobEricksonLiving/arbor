Application Diagram

To expand a node double click on it.

To remove an expanded node double click on it. Red nodes cannot be removed.

To see the node information right click it.

To generate the servers for a specific node, type the node's name and the server type(development, test, production). Then click the expand button.
Example: AX production

To delete a node's servers enter the node's name followed by the server type(development, test, production). Then click the remove button.
Example: AX production

ADDITIONAL COMMANDS
Expand Button Commands:
-expand all applications
Remove Button Commands:
-remove all applications
