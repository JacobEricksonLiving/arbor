APPLICATION DIAGRAM

To see the node information right click it.

To expand an application node double click on it.

To remove an expanded applicaton node double click it. To remove a node with no connections double click it.
NOTES: Nodes can be dragged. Red nodes cannot be removed.

To generate or remove application nodes(blue) you can type the node's name and click the appropriate button.
To generate or remove server nodes for a specific application, type the applications's name followed by the server type(production or pre-production). Then click the appropriate button. Example: AX production

Server Color key:
Application: green, Database: purple, Web: brown. Light-->Dark: Development-->Test-->Production

To generate edge labels type source-->target.
Example:Vision-->AX

ADDITIONAL COMMANDS
Generate Button Commands:
-generate all applications
-generate all edge labels
Remove Button Commands:
-remove all applications
